# Research Sources

FocusFlow was built from a small set of productivity and cognition notes about timing, motivation, and recovery.

## Short summary

- Circadian and ultradian rhythm alignment: work blocks should fit biological energy cycles instead of forcing a flat schedule.
- Pomodoro and ultradian rhythm ideas: short rest cycles help preserve attention and reduce mental drift.
- Pre-task boredom / under-stimulation: a brief low-input ramp can make deep work easier to enter.
- Motivation spillover: interest in one task can affect performance on others, so the app tries to reduce unnecessary switching.
